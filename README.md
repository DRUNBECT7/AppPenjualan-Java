Repository ini berisi proyek Aplikasi Penjualan Ringan yang dibangun menggunakan Java dan NetBeans IDE sebagai contoh aplikasi kasir sederhana berbasis desktop. Aplikasi ini menggunakan antarmuka Java Swing, sehingga mudah dipahami dan cocok untuk pembelajaran dasar pembuatan aplikasi desktop.

Proyek ini dirancang untuk membantu proses pencatatan transaksi sederhana, mulai dari pengelolaan data barang, input jumlah pembelian, perhitungan total otomatis, hingga menampilkan nota ringkas. Struktur kode dibuat sesederhana mungkin agar mudah dipelajari oleh mahasiswa, pemula Java, maupun pengembang yang ingin memahami logika dasar sistem penjualan. Aplikasi ini juga mendukung integrasi ke MySQL untuk pengembangan lanjutan.

Repository ini dapat digunakan sebagai media pembelajaran pemrograman Java, studi kasus GUI, hingga dasar pembuatan aplikasi kasir mini. Proyek terbuka untuk dikembangkan lebih jauh, baik dengan menambah fitur, memperbaiki tampilan, maupun meningkatkan sistem database.
